# Project Overview

This repository contains multiple projects. Below are the instructions to start each of them locally.

## Prerequisites

Ensure you have [Node.js](https://nodejs.org/) installed on your machine.
(Recommended version: LTS)

## Projects

### 1. bäko-copilot-demo

A React + Vite application.

**To start locally:**

1.  Navigate to the project directory:
    ```bash
    cd "bäko-copilot-demo"
    ```
2.  Install dependencies:
    ```bash
    npm install
    ```
3.  Start the development server:
    ```bash
    npm run dev
    ```
    The application will be available at `http://localhost:3000` (or `http://0.0.0.0:3000`).

## General Notes

- All projects use `vite` for development.
- Use `npm run build` to build the projects for production.
- Use `npm run preview` to preview the production build locally.
